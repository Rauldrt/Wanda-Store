# Wanda Store

Sistema de gestión inteligente y tienda online para Wanda Distribuidora.

## Características

- Panel de Administración (Clientes, Pedidos, Productos, Logística, Ajustes)
- Tienda Online para Clientes
- Sincronización en tiempo real con Firebase
- Diseño moderno y responsivo con Framer Motion y Tailwind CSS

## Tecnologías

- Next.js 14
- Firebase (Auth, Firestore, Storage)
- Tailwind CSS
- Lucide React (Iconos)
- Framer Motion (Animaciones)

## Instalación

1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Configura variables de entorno en `.env.local`
4. Ejecuta: `npm run dev`
